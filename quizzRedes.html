Fundamentos de redes

Competencias: Desarrollar soluciones tecnológicas para entornos Web mediante fundamentos de programación orientada a objetos, base de datos y redes de área local que atiendan las necesidades de las organizaciones.
Unidades: 6

Fundamentos de redes U1.Introducción a las Redes de Datos

implementará redes de área local realizando configuración de routers y switches, aplicando esquemas de direccionamiento IP para proporcionar acceso a recursos y servicios.ios.
Componentes de una red de Datos Clasificaión y evolucuión de las redes de datos 2 Clasificaión y evolucuión de las redes de datos

Fundamentos de redes U2.Protocolos y Comunicación en la Red

El alumno identificará el proceso de comunicación a través del reconocimiento de las reglas, protocolos y estándares existentes para describir el flujo de información en una red.
Protocolos y estándares Modelos de referencia Encapsulamiento de datos

Fundamentos de redes U3.Acceso a la Red

El alumno reconocerá las funciones de la capa física y de enlace de datos para la integración de dispositivos en una red de área local.
Control de acceso al medio Protocolo Ethernet switches LAN

Fundamentos de redes U4.Capa de Red

El alumno identificará los componentes que operan en la capa de red para la interconexión de redes de área local.
Protocolos de la capa de red Interconexión de redes Routers y onfiguracion basica de un router

Fundamentos de redes U5.Protocolo de internet

El alumno describirá los protocolos, funciones y servicios utilizados en las capas superiores del modelo de referencia OSI para la interacción entre las aplicaciones del usuario final.
Direccionamiento IPv4. Direccionamiento IPv6 1. Direccionamiento IPv6 2.

Fundamentos de redes U6. Capa de transporte y capa de aplicación

El alumno describirá los protocolos, funciones y servicios utilizados en las capas superiores del modelo de referencia OSI para la interacción entre las aplicaciones del usuario final.
Capas superiores. Protocolos y servicios de la capa de aplicación. Protocolos y funciones de la capa de transporte.
